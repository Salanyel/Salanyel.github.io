Kouji

iOS | C# | Augmented reality (Vuforia & ARKit) | Unity3D

Kouij is an augmented reality messaging application. The user can create his own avatar and uses it to shows its mood to his friends.
This application has been featured on the AppleStore!

This project is the second application created by Spooklight studios. This app let you Create a 3D avatar and send movie clip using Augmented reality to create a context with your environement. Your avatar will have many animations to communicate your feelings, humor, or other appropriate meaning you want to share with your friends.

On this project I have been the lead programmer. I worked with 7 developers, 1 artist director, 1 character designer and 4 animators.

Core: The augmented reality technologies used in the application depends on the phone and its version. I created a complete system to instantiate dynamically the correct SDK and linked it to the rest of the application. In case, we want to add another SDK, there is only one class to create to maintain it. The class that rules the application and defined all states that command the application has been defined and developed by myself.

AR: I was in charge of all features based on the augmented reality. Here is some examples: Marker taking, placing the character in the environment, simulating a position in the real environment of the user.

UI/Interaction: As a lead, I worked on the logic for the different transition between the screens and several other logic about the item instantiating and 3D rendering in the menu. The most difficult thing has been to manipulate the character on a 2D plane based on the gyroscope of the phone and simulates these input in the 3D environment of the AR SDK. The axes where never corresponding so I had to manage global and local positions.

Project: As a lead, my main responsibility was to be the reference for all of my colleagues. I answered to the application integrity questions and all others demands for the architecture. I was also responsible for the production of the version for the different stores we are targeting.

Storyfab

iOS | C# | Augmented reality (Vuforia) | Unity3D

Storyfab is an iOS application who helps the user to create short films by using augmented reality.
This application has been featured on the AppleStore!

This project was the idea which created Spooklight studios. In this application, you take a real image as the support for the augmented reality. When the studio has been founded, I was the only developer. I had to learn how to merge the new technologies together to record films in augmented reality. By being the most educated with Unity in our team, I also take in charge the pipeline of assets integration. In addition, I have worked on all aspect of the product :

UI: All UIs were designed by our art director. I have integrated them and animated them for every menu and interactions.

Gameplay: To film a video, the application uses a timeline. I created a complete time manager that allow the user to go back on time to an exact frame. All informations are saved : positions, rotations, animations, objects, states.

Core: The challenge was to use a AR technology with a video recording one in order to merge film parts. The first version of Storyfab is reliant on the AR solution. The avatar window show a complete problematic of the application : everything have to be customizable. I have created a character manager that can be expanded in the future with new features.

Tool: To help the artistic team of the studio to integrate assets more easily, I developed several tools to facilitate their works. For example, one tool is specialized to create staging for the several themes we have.

Hexmap tool generator

Unity3D | C# | Mesh creation | Shader | Procedural generation | Data managment

Hexmap tool generator is a tool designed to create 3D hexmap. The user have a flat surface in front of him and can add different features to the map before exporting it.

All elements linked to the map: ground, cliffs, rivers, water; are mesh generated from scratch. An algorithm calculates the position of each vertex and refreshes the corresponding mesh to display it.

Rivers and waters have their own shader to create a fluid movement and a fluid displacement.

All features (like urban, farm and plant) creation are based on procedural creation. Their rotation, numbers and type are selected by a parameter generation and a seed for each hexcell.

Incoming developments: Walls, towers, building, bridges, saving & loading feature.

Nested Prefab Plugin for Unity

C# | Tooling | Unity3D

Unity has a concept to manage a "template of object", it is called "Prefab". In Unity, there is a problem: when a prefab is contained inside a prefab, the engine does not manage it very well and end by losing data or scenes.

The Nested Prefab plugin offers a full solution to compile and uncompile prefabs contained in another one. It allows the user to save its project automatically and manage the integrity of its prefabs. The plugin also handles cascade behaviour to manage a recursive prefabs container.

The project is available on the Unity asset store.

Plugin on the Unity Asset Store
